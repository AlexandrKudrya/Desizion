# src/ - Исходный код приложения

## Файлы

| Файл | Описание |
|------|----------|
| `App.tsx` | Главный компонент приложения. Определяет роутинг между HomePage и ProjectPage |
| `main.tsx` | Точка входа. Рендерит App в DOM |
| `index.css` | Глобальные стили, Tailwind директивы, CSS переменные |

## Папки

| Папка | Описание |
|-------|----------|
| `components/` | React компоненты (UI, layout, фичи) |
| `lib/` | Утилиты, бизнес-логика, хелперы |
| `store/` | Zustand store для глобального состояния |
| `types/` | TypeScript типы и интерфейсы |
| `pages/` | Страницы приложения |
