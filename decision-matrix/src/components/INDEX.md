# components/ - React компоненты

## Структура

```
components/
├── ui/           # Базовые UI компоненты (shadcn/ui)
├── layout/       # Компоненты разметки (Header, Sidebar, Layout)
├── project/      # Компоненты для работы с проектами
├── criteria/     # Компоненты для критериев
├── alternatives/ # Компоненты для альтернатив
└── results/      # Компоненты для отображения результатов
```

## Подпапки

| Папка | Описание |
|-------|----------|
| `ui/` | Переиспользуемые UI примитивы: Button, Input, Dialog, Select, Tabs, Card, Badge, etc. Основаны на Radix UI |
| `layout/` | Layout-компоненты: Header (шапка), Sidebar (боковое меню), Layout (обёртка) |
| `project/` | Работа с проектами: список, карточка, заголовок с действиями |
| `criteria/` | Управление критериями: таблица, диалог создания/редактирования, ввод параметров нормализации |
| `alternatives/` | Управление альтернативами: таблица, диалог, ввод значений по критериям |
| `results/` | Отображение результатов: рейтинговая таблица, детальная разбивка баллов |
