# alternatives/ - Компоненты для альтернатив

## Файлы

| Файл | Описание |
|------|----------|
| `AlternativesTable.tsx` | Таблица альтернатив с динамическими столбцами (по одному на каждый критерий). Показывает название, значения по критериям, кнопки редактирования/удаления |
| `AlternativeDialog.tsx` | Диалог создания/редактирования альтернативы. Поля: название, описание, значения по всем критериям |
| `CriterionValueInput.tsx` | Поле ввода значения для конкретного критерия. Для категориальных - Select, для остальных - Input number |

## Динамическая таблица

```
┌───────────────┬──────────┬──────────┬──────────┬──────────┐
│ Название      │ Критерий1│ Критерий2│ Критерий3│ Действия │
├───────────────┼──────────┼──────────┼──────────┼──────────┤
│ Альтернатива 1│   100    │   50     │  Хорошо  │  ✏️ 🗑️   │
│ Альтернатива 2│   80     │   70     │  Плохо   │  ✏️ 🗑️   │
└───────────────┴──────────┴──────────┴──────────┴──────────┘
```

## Типы ввода значений

| Тип критерия | Компонент ввода |
|--------------|-----------------|
| Числовые (LINEAR, INVERSE, THRESHOLD, EXPONENTIAL) | `<Input type="number">` |
| Категориальные | `<Select>` с вариантами из categories |

## Использование

```tsx
<AlternativesTable
  projectId={projectId}
  alternatives={project.alternatives}
  criteria={project.criteria}
/>
```
