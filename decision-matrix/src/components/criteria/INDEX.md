# criteria/ - Компоненты для критериев

## Файлы

| Файл | Описание |
|------|----------|
| `CriteriaList.tsx` | Таблица критериев с колонками: Название, Вес, Тип нормализации, Действия. В футере показывает сумму весов с индикатором (100% = ок) |
| `CriterionDialog.tsx` | Диалог создания/редактирования критерия. Поля: название, вес, тип нормализации, параметры, описание |
| `NormalizationParamsInput.tsx` | Динамические поля параметров в зависимости от типа нормализации |

## Типы нормализации и их параметры

| Тип | Параметры |
|-----|-----------|
| LINEAR | min, max |
| INVERSE_LINEAR | min, max, threshold |
| THRESHOLD | min, max, threshold1 |
| EXPONENTIAL | min, max, threshold1, k (коэффициент спада) |
| CATEGORICAL | массив категорий [{label, value}] |

## Валидация весов

- Сумма весов всех критериев должна быть = 100%
- Визуальный индикатор: зелёная галочка если ок, оранжевый восклицательный знак если нет
- Вес каждого критерия: 1-100%

## Использование

```tsx
<CriteriaList projectId={projectId} criteria={project.criteria} />
```
