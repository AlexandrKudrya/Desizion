# layout/ - Компоненты разметки

## Файлы

| Файл | Описание |
|------|----------|
| `Header.tsx` | Шапка приложения. Содержит логотип, кнопки "Импорт" и "Новый проект" |
| `Sidebar.tsx` | Боковая панель со списком проектов. Показывает активный проект, позволяет переключаться между проектами |
| `Layout.tsx` | Главная обёртка. Объединяет Header, Sidebar и контентную область |

## Структура Layout

```
┌─────────────────────────────────────────┐
│                 Header                   │
├──────────┬──────────────────────────────┤
│          │                              │
│ Sidebar  │         Content              │
│          │       (children)             │
│          │                              │
└──────────┴──────────────────────────────┘
```

## Использование

```tsx
import { Layout } from '@/components/layout/Layout'

function App() {
  return (
    <Layout>
      <ProjectPage />
    </Layout>
  )
}
```
