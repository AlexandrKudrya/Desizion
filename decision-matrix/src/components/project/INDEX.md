# project/ - Компоненты для работы с проектами

## Файлы

| Файл | Описание |
|------|----------|
| `ProjectList.tsx` | Список всех проектов в виде сетки карточек. Показывается на главной странице. Содержит кнопку создания нового проекта |
| `ProjectCard.tsx` | Карточка одного проекта. Отображает название, описание, дату, количество критериев/альтернатив. Кнопка удаления с подтверждением |
| `ProjectHeader.tsx` | Заголовок открытого проекта. Содержит кнопку "Назад", название, кнопки "Экспорт" и "Удалить" |

## Функциональность

### ProjectList
- Отображает пустое состояние, если проектов нет
- Карточки в сетке (1-2-3 колонки в зависимости от ширины экрана)
- Клик по карточке открывает проект

### ProjectCard
- Показывает название и описание проекта
- Бейджи с количеством критериев и альтернатив
- Дата последнего обновления
- Удаление с AlertDialog подтверждением

### ProjectHeader
- Кнопка "Назад" для возврата к списку
- Экспорт проекта в JSON файл
- Удаление проекта
