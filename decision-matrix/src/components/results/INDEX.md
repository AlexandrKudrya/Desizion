# results/ - Компоненты результатов

## Файлы

| Файл | Описание |
|------|----------|
| `RankingTable.tsx` | Рейтинговая таблица альтернатив. Показывает место (с медалями для топ-3), название, общий балл в процентах, прогресс-бар |
| `ScoreBreakdown.tsx` | Детальная разбивка баллов для выбранной альтернативы. Таблица: Критерий, Исходное значение, Нормализованное, Вес, Вклад в итоговый балл |

## RankingTable

Визуализация рейтинга:
- 🏆 1 место - золотой фон
- 🥈 2 место - серебряный фон
- 🥉 3 место - бронзовый фон
- Прогресс-бар показывает % от лучшего результата

```
┌────────────────────────────────────────────┐
│ 🏆  Квартира на Приморской         85.2%  │
│     ████████████████████░░░░░             │
├────────────────────────────────────────────┤
│ 🥈  Квартира на Невском            72.1%  │
│     ██████████████░░░░░░░░░               │
└────────────────────────────────────────────┘
```

## ScoreBreakdown

Детальная таблица для анализа:

| Критерий | Значение | Норм. | Вес | Вклад |
|----------|----------|-------|-----|-------|
| Цена | 45000 | 80% | 30% | 24% |
| Площадь | 55 | 70% | 25% | 17.5% |
| ... | ... | ... | ... | ... |
| **Итого** | | | | **85.2%** |

## Использование

```tsx
<RankingTable alternatives={alternatives} criteria={criteria} />
<ScoreBreakdown alternative={selectedAlt} criteria={criteria} />
```
