# pages/ - Страницы приложения

## Файлы

| Файл | Описание |
|------|----------|
| `HomePage.tsx` | Главная страница со списком проектов. Показывается когда проект не выбран |
| `ProjectPage.tsx` | Страница проекта с вкладками: Критерии, Альтернативы, Результаты |

## Роутинг

Простой роутинг через `currentProjectId`:

```tsx
// App.tsx
function App() {
  const currentProjectId = useStore(state => state.currentProjectId)

  return (
    <Layout>
      {currentProjectId ? <ProjectPage /> : <HomePage />}
    </Layout>
  )
}
```

## HomePage

- Отображает `<ProjectList />` с карточками проектов
- Пустое состояние с призывом создать первый проект
- Максимальная ширина контента: 5xl

## ProjectPage

Структура:
```
┌─────────────────────────────────────────┐
│ ProjectHeader (название, экспорт, удалить)│
├─────────────────────────────────────────┤
│ [Критерии] [Альтернативы] [Результаты]  │ <- Tabs
├─────────────────────────────────────────┤
│                                         │
│           Контент вкладки               │
│                                         │
└─────────────────────────────────────────┘
```

### Вкладки

| Вкладка | Содержимое |
|---------|------------|
| Критерии | `<CriteriaList />` - таблица критериев с CRUD |
| Альтернативы | `<AlternativesTable />` - таблица альтернатив |
| Результаты | `<RankingTable />` + `<ScoreBreakdown />` с выбором альтернативы |
